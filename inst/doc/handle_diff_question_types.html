Survey Question Types

So you’ve gotten the API all setup and now you need to plot data from different types of questions. In this vignette, we will focus on the matrix and side-by-side (SBS) matrix type.

NOTE: it’s not recommended to use SBS matrices given the high cognitave load. It makes taking a survey harder for your participants. With that said, we used them ourselves before knowing better and want to ensure things don’t break on any survey that you process! So support is provided here to work with them!

Note below that the matrix question type has more complex column headers. This structure represents the question number Q46, the column number or matrix number 1 and then the sub question number _3.

So

Q46.1_1

represents question 46 (as labeled in your survey) which has the unique id QID54 in qualtrics. .1 represents matrix number one in a side by side matrix and _1 represents the first question in that matrix.

Now you can plot! Below we plot responses for each matrix in it’s own ggplot facet.

# it could be nice to calculate a percentage to with some function or argument
q54 %>%
  group_by(quest_text, choice_text, matrix_num) %>%
  count() %>%
  ggplot(aes(x = choice_text, y = n)) +
  geom_bar(stat = "identity") +
  facet_wrap(~matrix_num, nrow = 2) +
  labs(title = "How Often You Use a Tool",
       x = "Frequency of Use",
       y = "Count")

If you turn factors ON then it will return a list of data.frames - one data frame for each col in a SBS matrix.

# here, we just grab a df that has the data stacked in a way that is easily plottable
q55 <- get_question_resp(my_survey_ob$questions$QID55,
                         choice_factor = TRUE)
## [1] "You are trying to process a side by side matrix that have multiple sets of choices. I've created a list containing a df for each matrix in the sbs matrix."
# it could be nice to calculate a percentage to with some function or argument
q55[[1]] %>%
  group_by(quest_text, choice_text) %>%
  count() %>%
  ggplot(aes(x = choice_text, y = n)) +
  geom_bar(stat = "identity") +
  labs(title = "How Often You Use a Tool\n Factors turned on",
       x = "Frequency of Use",
       y = "Count")
